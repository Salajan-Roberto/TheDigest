# The Digest — Frontend

Interfața React a platformei The Digest.

Pentru documentație completă, setup și descrierea proiectului vezi [README-ul principal](../README.md).

## Rulare locală

```bash
npm install
npm run dev
```

Aplicația pornește la [http://localhost:5173](http://localhost:5173). Asigură-te că MetaMask este instalat și setat pe rețeaua Sepolia.
